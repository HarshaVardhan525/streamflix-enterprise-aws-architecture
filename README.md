# StreamFlix Enterprise AWS Architecture

A highly available and scalable StreamFlix web application deployed on AWS using Multi-AZ architecture, Application Load Balancer, Auto Scaling Groups, Launch Templates, NAT Gateway, and EC2.

## Architecture Components

- Amazon VPC
- Public Subnets
- Private Subnets
- Internet Gateway
- NAT Gateway
- EC2 Instances
- Amazon Machine Image (AMI)
- Launch Template
- Application Load Balancer
- Target Group
- Auto Scaling Group
- CloudWatch Scaling Policy
